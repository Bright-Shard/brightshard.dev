About HiddenHID

HiddenHID is a Tkinter-based and cross-platform tool for HID attacks. It renders as an invisible Tkinter window, but acts like a terminal, running commands or shortcuts invisible in the background.


OS Support

HiddenHID supports Windows primarily, with additional features for macOS, and just bare functionality on Linux (There's too much variety in Linux distros for reliable results on it).

Windows Features

Note: I only test Windows attacks on Windows 10. Please report Windows 11 bugs on GitHub, as I won't find them.

HiddenHID stays invisible and hides it's icon from the application tray/dock. The only way to tell that it's running on Windows is with the Task Manager. In addition, because it's precompiled as an EXE, HiddenHID can be launched from the Run menu, without ever opening a terminal.

macOS Features

HiddenHID pretends to be the Finder application (Sharing the name and toolbar elements), and also launches in fullscreen mode to hide the dock. On launch, HiddenHID will make all terminals invisible, hiding any evidence that it was launched, and force-quits all terminals when it closes to ensure it goes unnoticed.


Shortcuts

HiddenHID has a few pranks by default - to change the system wallpaper, for example - and even a reverse shell with Shortcuts, short keywords that make previously complex or long HID attacks much faster and hard to notice.
