What is Cookie Clicker++?

Cookie Clicker++ (CCPP for short) is a userscript you can use for hacking Cookie Clicker. It's mostly just for fun or to impress your friends, because it can get a little bit out of control...

I think the building count on the right can speak for itself. This userscript is pretty cracked.

How do I install CCPP?

This is a userscript, which means you'll need to have a userscript manager installed. I personally use Tampermonkey, but if you have another favourite userscript manager, it'll work with that too. After you have one installed, all you need to do is click the 'Install CCPP' button below - it will download the userscript, and Tampermonkey will prompt you to install it. All the links are below:

Install Tampermonkey for FireFox Install CCPP Tampermonkey's Website

The install prompt should look like this, except it'll say 'Install' instead of 'Reinstall'.

Using CCPP

Right now, CCPP runs from the JavaScript console in your browser. I might add buttons/a GUI later. To open it, right-click the page and hit 'Inspect Element', then switch to the 'Console' tab. On Firefox, you can also hit 'F12' to open that menu. You'll see a help menu showing you the various commands CCPP provides.
Also, CCPP will automagically update in the background, so don't worry about updating manually.
